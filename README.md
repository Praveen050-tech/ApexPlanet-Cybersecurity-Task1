# ApexPlanet Cybersecurity Internship — Task 1: Reference Ledger

## 🐧 Essential Linux Command Cheat Sheet
This reference card documents basic administration operations performed within my isolated virtual laboratory environment:

### Directory Navigation & Mapping
* `pwd` — Confirms your exact working path location.
* `ls -la` — Lists all storage file attributes, sizing, ownership metrics, and hidden files.

### Security and Privilege Controls
* `touch filename.txt` — Spawns an empty file pointer.
* `chmod 600 filename.txt` — Locks down read/write access strictly to the owner user.
* `sudo command` — Forces system commands to execute with elevated administrative root permissions.

### Network Diagnostics
* `ifconfig` or `ip a` — Lists active network interfaces and bound local IP addresses.
* `ping -c 4 [Target_IP]` — Transmits 4 ICMP echo request packets to verify device availability.
